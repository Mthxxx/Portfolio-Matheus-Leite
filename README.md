## 🎨 Design

<p>O visual foi pensado no estilo <strong>dark editorial luxury</strong>:</p>

<p>🖋️ <strong>Tipografia:</strong> Fraunces (display) + Syne (corpo) + DM Mono (código/labels)</p>
<p>🎨 <strong>Paleta:</strong> fundo quase preto <code>#0a0a08</code> · dourado <code>#c9a84c</code> · neon <code>#39ff8f</code></p>
<p>✂️ <strong>Layout:</strong> assimétrico, grid editorial, elementos sobrepostos</p>
<p>🎬 <strong>Animações:</strong> fadeup no carregamento, scroll reveal, cursor com lag suave</p>
